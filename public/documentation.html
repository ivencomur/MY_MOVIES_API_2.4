Get All Movies

Returns a list of all movies with details from MytopMovies.

  • Endpoint URL: /movies
  • Method: GET
  • Query Parameters: None
  • Request Body: None
  • Response Data: JSON array of movie objects (e.g., [{ "title": "Requiem for a Dream", "year": 2000, "description": "...", "genre": "...", "director": "...", "imageEndpoint URL": "..." }, ...])

Get movie by title

Shows description, genre, director, and image Endpoint URL for a single movie by title from MytopMovies.

  • Endpoint URL: /movies/:title
  • Method: GET
  • Query Parameters: None
  • Request Body: None
  • Response Data: JSON object containing movie details (e.g., { "title": "Requiem for a Dream", "year": 2000, "description": "...", "genre": "...", "director": "...", "imageEndpoint URL": "..." })
